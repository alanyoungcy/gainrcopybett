# Gainr - Sports Copy Trading Platform

A modern sports prediction market copy trading platform where users can follow and automatically copy bets from expert sports traders.

## Features

- 🎯 **Copy Trading** - Follow top-performing sports bettors
- 📊 **Live Markets** - Real-time sports betting markets from Soccer, Basketball, Tennis
- 👥 **Trader Profiles** - Detailed stats including ROI, win rate, and streaks
- 🎨 **Modern Design** - Clean, professional interface with orange theme
- 📱 **Responsive** - Works on all devices

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
